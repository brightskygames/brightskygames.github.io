# このアプリについて
このゲームは数字を並べる論理パズルである。マスにはヒント数字があり、これをもとにし、1,2,3,…の数字を縦横に順番に並べるようにする。斜めに並べてはいけない。

# 内容：
- ５００以上のレベル
- 難易度５つ

# その他機能：
- 間違っている数字を表示
- 自動的にセーブ
- ベストタイムの記録
- Game Centerのアチーブメントに挑戦できる
